# Sundar-Hotel-Resort-Main-F-B
# 🏨 Hotel CMS Website

## 📌 Project Overview

This is a fully functional **Hotel Website with CMS-like system** built using Antigravity.
The project allows users (clients) to easily **add, edit, and manage content** without coding knowledge.

---

## 🚀 Features

* 🏨 Dynamic Room Listing System
* ✏️ Easy Content Editing (No Coding Required)
* ➕ Add / ❌ Remove Rooms & Sections
* 🖼️ Image Update Support
* 📱 Fully Responsive Design
* ⚡ Fast Loading Website

---


---

## 📂 Project Structure

* `/index.html` → Main Website
* `/cms-panel` → Content Management Section
* `/assets` → Images & Media
* `/styles` → CSS Files

---

## 💼 Use Case

This project is ideal for:

* Hotels & Resorts
* Restaurants
* Small Businesses
* Freelance Website Selling

---

## ⚠️ Limitations

* No real database (Manual CMS system)
* No login authentication
* Content updates are manual
